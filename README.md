# chat-app
chat app using socket-io and Node.JS

##Features:
    1-public chat room 
    2-create rooms and chat on every room indvidual

##Requirements
    Node v0.12.*

##First Step
    Run `npm install`

##To run Node, Socket.io Servers
###Note: Server run on port 3000
    Run `node index.js`

##Now
    Go to `localhost:3000` and enjoy chating
